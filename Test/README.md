# Prueba tecnica: Sisorg Minimal web api ASP.NET React

## Descripción

Este repositorio contiene mi solución para la prueba técnica como parte del proceso de selección para Sisorg. El proyecto está desarrollado utilizando las tecnologías ASP.NET y React, y demuestra mis habilidades en el desarrollo full-stack y la integración de estas tecnologías.

## Características

- Aplicación web desarrollada con ASP.NET y React, que muestra mi capacidad para trabajar en ambas partes del stack.
- Implementación de ChartJS y React-Chartjs para el manejo de datos sobre un grafico.
- Uso de Bootstrap y Reactstrap para el manejo de estilos.

## Instrucciones de Ejecución

1. Clona este repositorio en tu máquina local.
2. Abre la solución en Visual Studio.

(Si es necesario navege a la carpeta `ClientApp/` y ejecute `npm install` para instalar las dependencias.)
